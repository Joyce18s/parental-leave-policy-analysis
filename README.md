# Parental Leave Data Analysis



## Objective



The goal of this project is to analyze parental leave policies across companies and identify trends in paid and unpaid maternity and paternity leave.



## Dataset



The dataset contains company-wise information about:



\* Industry

\* Paid Maternity Leave

\* Unpaid Maternity Leave

\* Paid Paternity Leave

\* Unpaid Paternity Leave



## 🧹 Data Cleaning



\* Checked missing values

\* Removed blank columns

\* Identified large gaps in paternity leave reporting



## 📈 Exploratory Data Analysis (EDA)



The analysis includes:



\* Distribution of paid maternity leave

\* Paid vs unpaid maternity leave comparison

\* Industry-wise leave analysis

\* Correlation analysis between paid and unpaid leave

\* Paternity leave availability insights



##🔍 Key Insights



\* Most companies offer 5–15 weeks of paid maternity leave

\* There is no strong correlation between paid and unpaid leave

\* Paternity leave data is missing for many companies

\* Leave policies vary significantly across industries
